This Repo contains my Projects on STM32F446RE Nucleo-64 Board.
Code is generated using CubeIDE from STM.

Project folders:

1. Numbered Folders contains all the Synchronisation primitives
2. Application and Booloader folder contains the Bare-Metal project of bootloader combined with FreeRTOS application
3. SM_* Contains the Functionality related to GPIOs, Timer (various modes like IC, OC, PWM), UART from the YT channel : https://www.youtube.com/playlist?list=PL_zvrXFdKgZrLsAkLo_1qPeW7cPOCyEiK
