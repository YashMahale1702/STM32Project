**STM32Project - STM32F446RE Nucleo-64 Development**

This repository contains a collection of projects and applications developed for the STM32F446RE Nucleo-64 microcontroller board. All code is generated and developed using STM32CubeIDE.

**Repository Structure**

The project is organized into the following sections:

1. **Numbered Folders** - Contain implementations of various synchronization primitives for concurrent programming and thread management.

2. **Application & Bootloader** - Features a bare-metal bootloader integrated with a FreeRTOS-based embedded application, demonstrating advanced firmware architecture.

3. **SM_* Modules** - Provide hands-on implementations of core STM32 functionalities including:
   - GPIO control
   - Timer operations (Input Capture, Output Compare, PWM modes)
   - UART serial communication
   
   These modules are based on the comprehensive tutorial series available at: https://www.youtube.com/playlist?list=PL_zvrXFdKgZrLsAkLo_1qPeW7cPOCyEiK
